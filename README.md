Script untuk xmlrpcbrute di linux atau termux

# INSTALL
```php
$ apt update && apt upgrade
$ apt install python
$ apt install figlet
$ apt install git
$ git clone https://github.com/herocan-7/xmlrpcbrute/
$ cd xmlrpcbrute
$ python -m pip install -r requirements.txt
$ python test.py
```
